Во вложенном файле лог данных с сенсора. Каждая строка в нем представляет из себя один отсчет.
Структура файла:
1. Колонка Offset - электронное смещение спектроанализатора.
2. Колонки Pow1...Pow80 - значения оптической мощности для каждого из участков спектра.
3. Колонки Wlen1...Wlen80 - значения длины волны для каждого из участков спектра.
4. Остальные колонки - технологические параметры, значения не имеют.
